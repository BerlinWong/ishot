# Leica Style Watermark Engine (Pro Edition)

这是一个从追求复刻到超越原生体验的“徕卡/苹果风格”水印引擎。它不仅仅是代码的堆砌，更是对移动端摄影工作流深度优化的结晶。

## 🚀 我们的演进之路 (Mental Journey)

这个项目的诞生始于对一张理想水印图（`demo.png`）的执着追求。在开发过程中，我们经历了一场从“能运行”到“极致丝滑”的蜕变：

### 第一阶段：复刻逻辑的诞生
最初，我们只是想把图片的 EXIF 信息提取出来，在底部拼上一个长条。但很快我们发现，传统的“图片上传 -> 处理 -> 返回”模式在 iOS 快捷指令上体验极差，数 MB 的 HEIC 图片在弱网下传输延迟巨大。

### 第二阶段：快捷指令的“元数据”革命
为了让水印生成达到“秒开”级别，我们重构了架构，引入了 **“无图/缩略图代理模式”**。快捷指令仅需发送几十字节的 JSON 元数据，后端即可在云端渲染出 100% 完整比例的高清晰度水印条，再回传至手机本地进行拼接。

### 第三阶段：像素级的“抗糊”战争
当用户在大图上查看水印时，JPEG 的压缩纹理和文字锯齿一度让人抓狂。为此，我们引入了 **“超采样渲染 (Supersampling)”**：
- 在内存中以 **200% 分辨率**进行渲染（4K 画布渲染 8K 细节）。
- 全链路切换为 **无损 PNG 格式**，确保边缘锐利如刀锋。

### 第四阶段：品牌与光线的“感知平衡”
- **品牌适配**: 针对索尼 (SONY) 等品牌，我们不仅更换了文字，还对字体的 **Baseline (基准线)** 进行了动态下沉补偿，解决了不同品牌文字与签名图标视觉不居中的难题。
- **环境感知**: 引入 APEX 亮度指数算法，水印颜色会跟随照片的明暗（夜景或强光）自动切换黑白主题。

## 🌟 核心黑科技

- **万能 JSON 解包器**: 自动兼容 iOS 快捷指令各种奇葩的嵌套结构和文本化降级行为。
- **动态坐标解析**: 集成高德 API，实现 [景区 > 地标 > 城市] 的递归级地址识别。
- **自适应比例引擎**: 无论 1200 万还是 4800 万像素，水印条均能实现等比例完美缩放。

## 🛠️ 安装与部署

```bash
pip install -r requirements.txt
# 配置 .env 中的 AMAP_KEY
uvicorn main:app --host 0.0.0.0 --port 8000
```

配合 iOS 快捷指令，您可以实现：
**选择照片 -> 秒出高清水印条 -> 垂直拼贴 -> (完成)**
